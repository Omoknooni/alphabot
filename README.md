# AlphaBot
Discord bot for python study

# Description
1. 간단한 명령어들을 포함하는 디스코드 봇입니다
2. 구동 전 같은 경로에 Token.txt을 만들어 봇 토큰을 저장
3. 아직 불안정한 명령어들이 다수 존재합니다 (ex. play)

# Commands
* 주사위 굴리기 `!roll [범위]`
* 유튜브 영상 재생 및 제어(mp3) `!play [유튜브 영상 링크]`
* 주식 종목 현황 확인 `!stock [종목명]`
